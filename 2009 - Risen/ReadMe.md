### Разное

- Файл ConfigUser.xml находится в папке C:\Users\*****\AppData\Local\Risen.
- Для нормализации громкости звуков и диалогов нужно добавить строчку Master="765" в файл ConfigUser.xml
```
<Audio
UseHardwareSounds="true"
>
<Volume
Ambient="255"
Effect="252"
Master="765"
Music="203"
>
</Volume>
</Audio>
```
